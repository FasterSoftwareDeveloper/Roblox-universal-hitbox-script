## üéÆ HOW TO USE

### BASIC SETUP
1. **Enable ESP** - See enemies through walls
2. **Enable Hitbox** - Make targets bigger
3. **Enable Aimbot** - Auto-aim at enemies
4. **Adjust FOV** - Set aim area size
5. **Choose Target** - Head or body targeting

### FREE CAMERA SYSTEM
1. Press **Shift + P** to enable Free Camera
2. Use **W/A/S/D** to move
3. **Right Click** or **Shift Lock** to look around
4. **Q/E** for up/down movement
5. Press **V** to unlock mouse if stuck
6. Press **Shift + P** again to disable

### TROUBLESHOOTING
1. **ESP not showing?** - Toggle ESP OFF then ON
2. **Aimbot not working?** - Check if mouse is connected
3. **Free Camera stuck?** - Press V to unlock mouse
4. **GUI not appearing?** - Press T or re-execute script
5. **Features resetting?** - Normal after respawn, re-enable
6. **Gray box in Free Camera?** - Fixed in V9.0

### FREE CAMERA TIPS
1. If mouse gets locked, press **V** to unlock
2. Use **Shift Lock** for hands-free looking
3. Camera moves independently from character
4. Character is hidden while Free Camera is active
5. All other features work normally in Free Camera
