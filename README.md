## üéÆ HOW TO USE

### BASIC SETUP
1. **Enable ESP** - See enemies through walls
2. **Enable Hitbox** - Make targets bigger
3. **Enable Aimbot** - Auto-aim at enemies
4. **Adjust FOV** - Set aim area size
5. **Choose Target** - Head or body targeting

### QUICK COMBOS

#### FOR EASY KILLS:
```lua
ESP: ON
Hitbox: ON
Aimbot: ON
FOV: 80
Mode: Crosshair
```

#### FOR HIGH MOBILITY:
```lua
Speed: ON (18+)
FlyJump: ON
Click Teleport: ON
```

#### FOR STEALTH:
```lua

```

#### FOR TEAM GAMES:
```lua

```
